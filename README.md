1. Install Dependencies
	--> gem install middleman
	--> gem install foreman
	--> npm install -g bower
	--> npm install -g grunt-cli
	--> npm install -g less
	
2. Running Static App
	--> cd PROJECT_DIRECTORY
	--> foreman start
	--> Listening on 0.0.0.0:4567
	--> CTRL+C to Stop
	
3. Exporting Static App
	--> cd PROJECT_DIRECTORY
	--> bundle exec middleman build
